# Core documentation

Project-local documentation for **erganis-core**. Platform-wide planning lives in the parent repo: [erganis/docs/erganis-product-plan.md](https://github.com/enabledtocreate/erganis/blob/master/docs/erganis-product-plan.md).

## Layout

| Path | Purpose |
|------|---------|
| **`temp/`** | Scratch and staging — notes, drafts, ADRs, and fragments until APM promotes them to managed docs (`.apm/templates/` → `ARCHITECTURE.md`, `PRD.md`, etc.) |

Do **not** treat `temp/` as the long-term source of truth. When APM is ready, content migrates out of `temp/` into generated or template-backed artifacts.
